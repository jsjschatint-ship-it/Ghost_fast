7. DNS 全记录枚举 / A / AAAA / MX / TXT / SOA / SRV / NS / CAA / CNAME / DMARC / SPF / DKIM

枚举常见 DNS 记录类型，整理邮件基础设施，并识别 TXT 中常见 SaaS 验证 token（GitHub / Google Workspace / AWS SES / Atlassian / 钉钉等）。与 dnsadv 的 AXFR 检查互补。

8. 敏感路径存在性 / .git/HEAD / .env / swagger / actuator / .DS_Store ...

对常见敏感路径做 HEAD / GET 存在性判断，仅记录状态、大小和前 1KB 摘要，不下载完整内容 / 不做利用验证。HTML 兜底页（SPA）会被启发式过滤。

URL 类别 严重度 状态 大小 摘要

9. CDN / WAF 指纹 + 源站候选发现 / Cloudflare / Akamai / 阿里云盾 / 腾讯云 / 雷池...

基于响应头、CNAME 和 DNS 信号识别 CDN/WAF，并通过 MX / SPF / 常见业务子域（mail / direct / origin / cpanel 等）生成源站候选。不做漏洞验证。

10. ASN / BGP 网段扩展 / bgpview.io + team-cymru

输入 IP / 域名 / AS13335 / 组织名，查询关联 ASN 并展开 BGP 前缀，输出 CIDR 供 3. portscan / 2. httpx 继续资产测绘。

11. WHOIS / RDAP 实时查询 / 注册商 / 创建过期 / 联系邮箱 / NS

同时查询 RDAP（JSON）和 WHOIS（plaintext），提取注册商、时间、NS 与联系人信息；可按注册邮箱/组织反查关联域名。

12. WebMeta 网页元信息 / title / meta / ICP / robots / sitemap

采集公开页面元信息、备案号、邮箱、电话，并解析 robots.txt / sitemap.xml 中公开列出的路径和 URL。只读取公开资源，不做登录、表单提交或漏洞验证。
